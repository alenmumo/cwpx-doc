Detailed Description

Request object for reading request data.

The Request object provides functions for getting values from the HTTP request. The values come in several ways, most of them as 'key-value' pairs. Functions apply these params:

Parameters
keyA name that identifies the requested data.
indexAn index for the requested data, in case there are many equal names.
attrAn attribute of the requested file.
scriptpathPath of the script for fordwarding the request.
Returns
A valid 'char *' containinig the value, NULL or 0|1 in case of returning 'int'.
