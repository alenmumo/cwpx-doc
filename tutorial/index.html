Páginas CWPX

Una página CWPX es un archivo especial en el cual se puede escribir código HTML con código C incrustado, haciendo que el contenido del HTML sea dinámico, con un estilo similar a los archivos PHP, ASP y JSP, utilizando etiquetas especiales para delimitar el códido dinámico y el código estático de la página.

iCWPX API

Este tutorial describe el funcionamiento básico de la API iCWPX, la cual es una interface que trabaja como un 'superset 'del marco de desarrollo web CWPX. Dicha API permite la implementación de patrones MVC y servicios REST de una manera sencilla.
